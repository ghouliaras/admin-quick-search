Add a powerful quick search feature to your WordPress admin dashboard for lightning-fast navigation.

== Description ==

Admin Quick Search enhances your WordPress admin experience by adding a universal search feature that allows you to quickly find and navigate to any content in your dashboard.

= Features =

* **Universal Search**: Search across posts, pages, users, and plugins from one place
* **Keyboard Shortcuts**: Use Ctrl+K (or Cmd+K on Mac) to instantly open the search
* **Fast Navigation**: Jump directly to edit screens without clicking through menus
* **Keyboard Navigation**: Use arrow keys to navigate results and Enter to select
* **Real-time Results**: See search results as you type with no page reload
* **Permission-Aware**: Only shows results the current user has permission to access
* **Admin Bar Access**: Quick access via the search icon in the admin bar
* **Dedicated Menu**: Access from the Quick Search menu in the admin sidebar

= Use Cases =

* Quickly find and edit a specific post or page
* Jump to user profiles without navigating through Users menu
* Find plugins by name instantly
* Access recent content with smart suggestions

== Installation ==

1. Upload the `admin-quick-search` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use Ctrl+K (Cmd+K on Mac) anywhere in the admin to start searching

== Frequently Asked Questions ==

= How do I open the search? =

Three ways:
1. Press Ctrl+K on Windows/Linux or Cmd+K on Mac
2. Click the search icon in the admin bar
3. Access from the Quick Search menu in the admin sidebar

= Can I customize what gets searched? =

Currently, the plugin searches posts, pages, users, and plugins. Future versions will include settings to customize search sources.

= Does it work with custom post types? =

Version 1.0 focuses on core WordPress content. Custom post type support is planned for version 1.1.
